Intallation : 

 1) git clone https://github.com/Core-Techs-Git/distrib_d8.git .
 2) Lancer  composer install
 3) installer votre site en créant la base de données, et suivre les étapes pour installer les composants (features)

Fonctionnalités par défaut (installées par défaut) : 

 1) Média avec les 3 provider (image, document , vidéo),
 2) Médiathèque (Possibilité d'intégrer un média dans le wysiwyg à partir de la médiathèque) ... il faut oublier IMCE
 3) Modules SEO (pathauto, rdirect, yoast_seo)  avec une configuration basique
 4) Configurations Wysiwyg
 5) Configurations divers ...

Fonctionnalités optionnelles (à activer pondant l'installation du site)

 1) Le Workflows  avec les étapes standard  et les notifications 
 2) Le multilingue
